# 两轮电动车终端市场调研

本次渠道调研我们访谈了电动车行业的11位专家，包括贵阳和上海的九号电动车经销商、台平科技股份有限公司的销售经理、安庆市中天数码技术有限公司的合伙人、上海普陀区雅迪电动车专卖店的店长、雅迪科技集团有限公司北京立水桥店的店长、台铃科技股份有限公司的区域销售经理、深圳的爱玛电动车一级经销商、上海的小牛电动车代理商、安徽合肥的九号电动车经销商以及小牛电动广东梅州的直营店店长，针对两轮电动车终端市场进行了调研，核心观点如下：

### 【专家背景介绍】
这些专家的背景各不相同，他们分别来自不同的地区和品牌。贵阳和上海的专家分别是九号电动车的经销商，而台平科技股份有限公司的销售经理则负责徐州、宿迁市场的销售工作，主要销售的品牌是台铃。安庆市中天数码技术有限公司的合伙人是九号电动车的总代理商，主要在安庆地区进行销售。上海普陀区雅迪电动车专卖店的店长和雅迪科技集团有限公司北京立水桥店的店长分别代表雅迪品牌。台铃科技股份有限公司的区域销售经理负责南昌地区的销售。深圳的专家是爱玛电动车的一级经销商，上海的专家是小牛电动车的代理商，而安徽合肥的专家是九号电动车的经销商。最后，小牛电动广东梅州的直营店店长主要负责小牛电动车的销售和推广。

### 【店内销量情况】
根据各地区专家的回答，贵阳九号电动车的经销商表示，上个月的销量是670台，比去年同期翻了一倍。而上海九号电动车的经销商则表示，本月销量比去年同期销量要大，但年度销量比往年都要少。台平科技股份有限公司的销售经理负责徐州、宿迁市场的销售工作，他们主要销售的品牌是台铃，他表示店内本月销量约为100-150台车，年度销量约为1800台左右。安庆市中天数码技术有限公司的合伙人，他所在的公司是九号电动车的总代理商，主要在安庆地区进行销售，他表示本月销量较上个月有所增长，年度销量比去年同期大概增长一倍左右。上海普陀区雅迪电动车专卖店的店长表示，本月销量还可以，年度销量为800台左右。雅迪科技集团有限公司北京立水桥店的店长表示，本月销量不到100辆，年度销量到目前为止还没到1000辆。

### 【店内库存和库销比】
店内库存和库销比的水平因地区和品牌的不同而有所差异。贵阳九号电动车的经销商表示，店内库存大概是300多台，库销比大概是销量的一半。而上海九号电动车的经销商则表示，库销比大概在80辆左右。台平科技股份有限公司的销售经理表示，他们在徐州、宿迁市场的店内库存的库销比大概保持在15%左右。安庆市中天数码技术有限公司的合伙人则表示，他们作为九号电动车的总代理商，库销比大概是200%左右。

### 【近期促销活动和降价水平】
近期的促销活动包括贵阳九号电动车的四个店同时开业、新品发布、开业50的活动以及邀请当地客户参加车友会，上海九号电动车的降价活动，台平科技股份有限公司在徐州、宿迁市场的广场活动，安庆市中天数码技术有限公司的暑期789公司的资金补贴，上海普陀区雅迪电动车专卖店的促销活动，雅迪科技集团有限公司北京立水桥店的参加抽奖和与京东同步价格的活动，台铃科技股份有限公司南昌区域的买车启动会、粉丝节、开学季和国庆节，深圳爱玛电动车的拿货价1400卖1399的一款车型和返利200块钱的活动，上海小牛电动车的多波次促销活动，以及安徽合肥九号电动车的线上线下价格统一活动和小牛电动广东梅州的618和8月30号的活动。

### 【近期发布的新款车型介绍】
近期发布的新款车型有多个品牌和型号。贵阳和安庆的九号电动车经销商表示，九号近期发布的新款车型有九号V30C和九号m max110P，其中V30C价格为2000多元，适合学生和宝妈使用。上海的九号电动车经销商则提到了F30C加长开裆的电动车，适合送外卖，空间较大，性价比较高，售价在3000到3600元之间。此外还有其他车型，如Q系列和V系列，以及即将发售的M110和M10，售价为7999元，优惠后到手价为7399元。安徽合肥的九号电动车经销商则提到了微系列和F mix两款新车型。

### 【新款上市后的热度和实际销售表现】
新款上市后的热度和实际销售表现因地区和品牌而异。贵阳九号电动车的经销商表示，新款上市后的热度很高，实际销售表现也很好，销量翻了一倍。然而，上海九号电动车的经销商则表示，虽然热度不错，但由于车辆还未到店，实际销售表现还无法评估。安庆地区的九号电动车总代理商则表示，新款车型的热度和实际销售表现还可以，但由于门店到货数量有限，销售量并不多。

### 【新国标换代的进展，电摩和电自各自的销售占比】
新国标换代的进展在各地区有所不同。贵阳九号电动车的经销商表示，虽然政府还未强制交换，但老百姓已经自发开始选购新国标的电摩车型。安庆市中天数码技术有限公司的合伙人则表示，目前整个路面的60%还是国标，60%还是电摩。上海小牛电动车的代理商表示，新国标换代的进展基本完成，上海地区已经完成了两次换购，淘汰了一些旧车型。而来自安徽合肥的九号电动车经销商表示，今年12月31号会去拍排，临时牌照会取消，但具体是否延期官方还未给出明确回复。

### 【两轮电动车销售均价情况】
根据调研结果，两轮电动车的销售均价因地区和品牌而异。贵阳和上海的九号电动车经销商表示，销售均价在5000元左右，但今年可能会有所降低。台平科技股份有限公司的销售经理表示，他们主要销售的品牌是台铃，其终端店内的销售均价是两千五左右，预计今年会降低10%-20%。安庆市中天数码技术有限公司的合伙人表示，他们的销售均价大约是3000元。上海普陀区雅迪电动车专卖店的店长表示，他们的销售均价大约是4000元。雅迪科技集团有限公司北京立水桥店的店长表示，他们的销售均价是3500多元，较去年的3900多元有所下降。深圳的爱玛电动车一级经销商表示，他们的销售均价是2000元左右，较去年的3000元以上有较大的变化。

### 【各价格带车型的占比情况】
电动车价格带占比情况因地区而异。贵阳九号电动车的经销商表示，2000以下的车型基本没有，2000到5000的车型占比大约为60%，5000以上的车型占比大约为40%。上海九号电动车的经销商则表示，两千以下的车型占比店里的30%，五千左右的车型占比40%，五千以上的顶配车型占比30%。台平科技股份有限公司的销售经理表示，他们在徐州、宿迁市场的销售情况是，2000以下的车型占比约为30%，2000到5000的车型占比约为30%，5000以上的车型占比很少，接近于零。安庆市中天数码技术有限公司的合伙人则表示，他们在安庆地区的销售情况是，2000以下的车型占比为0%，2000-5000的车型占比为20%，5000以上的车型占比为80%。

### 【当地退网、门店转让的经销商数量占比】
贵阳九号电动车的经销商表示，当地退网、门店转让的经销商数量占比不多，只有个别经销商进行门店转让，大部分经销商仍然坚持经营。然而在安庆地区，九号电动车的总代理商安庆市中天数码技术有限公司的合伙人表示，目前当地退网、门店转让的经销商数量为零，没有转让的经销商。相比之下，上海普陀区雅迪电动车专卖店的店长表示，当地退网、门店转让的经销商数量占比为70%。然而，台铃科技股份有限公司的区域销售经理表示，他没有统计过南昌地区的相关数据，所以无法提供当地退网和门店转让的经销商数量占比。

### 【经销商对下半年预期情况】
经销商对下半年的预期情况各有不同。贵阳九号电动车的经销商认为淡季可能会相对较好，因为新国标将于12月底到期，可能会有更多消费者前来购买车辆。然而，上海九号电动车的经销商对下半年的预期比较悲观，认为销售情况不会好转。台平科技股份有限公司的销售经理预期下半年的销售情况整体持平，而安庆市中天数码技术有限公司的合伙人对下半年的预期情况比较有信心。深圳的爱玛电动车一级经销商对下半年预期情况一般，认为九月份可能会好一点，但十月份下旬和十一月份生意会一般，而十二月份的维修业务和各方面可能会比较好。
